# اکستنشن Prettier

## توضیحات
Prettier یک ابزار فرمت کردن کد است که به شما این امکان را می‌دهد که کد خود را به یک فرمت استاندارد و خوانا تبدیل کنید. این ابزار برای برنامه نویسان JavaScript، TypeScript، HTML، CSS، و سایر زبان‌های مرتبط استفاده می‌شود.

## کارایی اصلی
- فرمت کردن خودکار کد با استانداردهای معتبر برنامه نویسی.
- پشتیبانی از فرمت کردن کد JavaScript، TypeScript، HTML، CSS و سایر زبان‌ها.
- قابلیت پیکربندی گسترده برای تطابق با سلیقه‌های شخصی.

## نحوه استفاده
1. ابتدا از [لینک دانلود](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) ابزار Prettier را در Visual Studio Code نصب کنید.
2. باز کنید یک فایل کد (مثلاً یک فایل JavaScript یا HTML).
3. از دستورات مختلف (مانند فشردن `Ctrl + Shift + P` و انتخاب "Format Document") برای استفاده از Prettier برای فرمت کردن کد خود استفاده کنید.

## نکات مهم
- برای تنظیمات دقیق‌تر، می‌توانید فایل `prettier.config.js` در پروژه خود ایجاد کرده و تنظیمات خاص خود را اعمال کنید.
- اطمینان حاصل کنید که پیش از استفاده از Prettier، پشتیبانی از زبان‌های مورد نظر خود را در تنظیمات Visual Studio Code فعال کرده باشید.

اکستنشن Prettier به شما کمک می‌کند تا کدتان همواره به شکل یکسان و استاندارد نگه داشته شود.
